# Фильмы

Приложение о фильмах с детальной информацией

## Языки, библиотеки, утилиты

- Retrofit
- Modularization
- Constraint layout
- MVVM
- Glide
- Coroutines
- Mockk
- Hilt
- Flow
- Paging3

## Архитектура

* Многомодульность
* Single Activity Application
* Clean Architecture
* MVVM
